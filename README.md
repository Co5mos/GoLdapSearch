# GoLdapSearch

查询 Ldap 信息

```
Usage: ldapsearch [-h host] [-u username] [-p password] [-b baseDn]

Options:
  -H string
    	ldap server host format: ldap[s]://hostname:port/ (default "ldap://192.168.1.1:389")
  -b string
    	base DN for search
  -p string
    	password for authentication
  -u string
    	username for authentication
```
